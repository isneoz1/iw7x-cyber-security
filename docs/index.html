Step 1 · Get Kali Linux

New to Kali? Pick a way in.

iw7x runs its tools on Kali Linux. All four methods below are free and official — choose whichever fits your setup, then install iw7x underneath.

Virtual Machine

easiest

Run Kali safely inside your current OS — no dual-boot. Install VirtualBox, import Kali's prebuilt VM image, boot (login kali / kali), then update once:

sudo apt update && sudo apt -y full-upgrade
Download the Kali VM

Windows, no VM (WSL2)

no VM

Run Kali as a Windows app. In an admin PowerShell, then launch Kali from the Start menu:

wsl --install
wsl --install -d kali-linux
Kali on WSL guide

Bare metal / dual boot

fastest

Best performance. Flash the Kali installer ISO to an 8 GB+ USB with balenaEtcher or Rufus, boot from it (F12 / Esc at startup), and run the graphical installer — you can install alongside your current OS.

Download the installer ISO

Live USB

try it

Want to test first? Flash the Kali Live image, boot from the USB and choose Live — everything runs from RAM, nothing touches your disk.

Download Kali Live

FAQ

Questions, answered.

Is iw7x legal to use?

iw7x is for authorized security testing and education only — systems you own or have explicit written permission to test, plus CTFs, labs and authorized engagements. Unauthorized access to systems is illegal, and you are solely responsible for how you use it.

What OS do I need?

Kali Linux is the recommended platform — bare-metal, in a VM, or Kali on WSL2 — that's where iw7x installs and launches tools. The menu, search, tags and advisor browse on any OS with Python 3.10+.

Does it really have 12,000+ tools?

Yes — 14,469 deduplicated tools across 52 categories ship in the catalog, and a built-in scanner keeps pulling new ones from BlackArch, Kali and dozens of awesome-lists.

How do I add a tool?

Add one JSON object to catalog.json and open a pull request — it's the easiest open-source contribution you'll ever make. See CONTRIBUTING.md.

Is it free?

100% free and open source under the MIT license. The individual tools keep their own respective licenses.
