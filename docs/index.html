Quickstart
# 1 · clone git clone https://github.com/isneoz1/iw7x-cyber-security.git cd iw7x-cyber-security # 2 · install deps (git, python, pipx, go…) chmod +x install.sh && ./install.sh # 3 · launch the arsenal python3 neoz.py

Running in 30 seconds

Recommended on Kali Linux (or Kali on WSL2). On first run iw7x fetches the full catalog so every tool is ready.

  1. ›
    Menu or shellLaunch the interactive menu, or run python3 neoz.py nmap to install & run a tool directly.
  2. ›
    Search & advisorType /query to search, or press R for "what do you want to do?".
  3. ›
    Keep growingRun --update once, or --watch to keep scanning for new tools forever.
