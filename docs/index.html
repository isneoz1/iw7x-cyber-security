FAQ

Questions, answered.

Is iw7x legal to use?

iw7x is for authorized security testing and education only — systems you own or have explicit written permission to test, plus CTFs, labs and authorized engagements. Unauthorized access to systems is illegal, and you are solely responsible for how you use it.

What OS do I need?

Kali Linux (or Kali on WSL2) is recommended — that's where iw7x installs and launches tools. The menu, search, tags and advisor work on any OS with Python 3.10+.

Does it really have 10,000+ tools?

Yes — 10,506 deduplicated tools across 44 categories ship in the catalog, and a built-in scanner keeps pulling new ones from BlackArch, Kali and dozens of awesome-lists.

How do I add a tool?

Add one JSON object to catalog.json and open a pull request — it's the easiest open-source contribution you'll ever make. See CONTRIBUTING.md.

Is it free?

100% free and open source under the MIT license. The individual tools keep their own respective licenses.
